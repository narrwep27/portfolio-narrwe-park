Book Keep

Book Keep was inspired by the GoodReads app. Being an avid reader, I wanted to build a project more personal to me. This app allows users to catalog books they want to read, are reading, or have read. I also connected this app to a 3rd party API (Open Library) so that users can search for books using an ISBN number.
